# TRPG Relation Maker

TRPGで使える相関図を、直感的に作成できるWebツール。

関係が絡み合う物語ほど、整理は難しい。  
だからこそ、一目で見える形に。

## ✨ できること

- キャラクター（ノード）の追加・編集
- キャラクター同士の関係を線で作成
- 相関図を視覚的に整理
- 画像として出力（PNG / JPEG）

---

## 📝 使い方

1. キャラクターを追加
2. 関係を線でつなぐ
3. 完成したら画像として保存
